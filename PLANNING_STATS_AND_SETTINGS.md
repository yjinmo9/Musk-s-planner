# 🚀 Musk's Planner: Stats & Settings Feature Plan

이 앱을 실제 "머스크 식"으로 극한의 효율성을 추구하며 사용한다고 가정했을 때, **Stats(통계)**와 **Settings(설정)**가 단순히 정보만 보여주는 것이 아니라 **행동을 교정하고 동기를 부여하는 도구**가 되어야 합니다.

다음은 사용자 관점에서 기획한 기능 명세입니다.

---

## 1. 📊 STATS (통계) - "Mission Control"

**목표:** 단순한 기록의 나열이 아니라, 나의 **생산성 엔진 성능**을 시각화하여 보여줍니다.

### 🟢 Free Tier (기본 원격 측정 / Basic Telemetry)

_누구나 자신의 하루를 돌아볼 수 있는 필수적인 피드백을 제공합니다._

1.  **Daily Launch Readiness (일일 완료율)**
    - **기능:** 오늘 설정한 'Main Things' 5개 중 몇 개를 완수했는지 보여주는 게이지.
    - **메시지:** "시스템 준비 80% 완료. 내일은 100%를 목표로 하십시오."
2.  **Weekly Momentum (주간 모멘텀)**
    - **기능:** 지난 7일간 수행한 업무량(Time Blocks 총 시간 or 태스크 수) 막대 그래프.
    - **목적:** 일관성 유지. 그래프가 끊기지 않도록 유도.
3.  **Time Allocation (시간 할당)**
    - **기능:** 오늘 하루 5가지 색상(카테고리)별로 시간이 어떻게 분배되었는지 보여주는 심플한 도넛 차트.
    - **예시:** Deep Work 50%, Meeting 20%, Health 10%...
4.  **Streak Counter (연속 실행)**
    - **기능:** 며칠 연속으로 플래너를 작성하고 하루를 마감했는지 표시.
    - **효과:** 앱 사용 습관 형성.

### 🟣 Pro Tier (유료 / "Neural Link" Edition)

_데이터를 통해 자신의 한계를 분석하고 최적화하려는 하드코어 유저를 위한 기능입니다._

1.  **Focus Quality Score (몰입 품질 점수)**
    - **기능:** 시간 블록의 파편화를 분석합니다. 10분짜리 업무가 산발적으로 흩어져 있으면 점수가 낮고, 긴 시간(Deep Work)이 확보되어 있으면 점수가 높습니다.
    - **인사이트:** "컨텍스트 스위칭이 너무 잦습니다. 업무를 묶어서 처리하세요."
2.  **Peak Performance Heatmap (피크 타임 히트맵)**
    - **기능:** 내가 가장 많은 업무를 완료(체크)한 시간대를 히트맵으로 시각화.
    - **가치:** "당신은 오전 9시~11시에 가장 생산적입니다. 이때 중요한 회의를 피하세요."
3.  **Plan vs. Reality (계획 대 실행 - 회고 분석)**
    - **기능:** 계획했던 시간 대비 실제 소요된 시간의 차이를 추적(별도의 '실행' 모드 필요).
    - **가치:** "당신은 항상 회의 시간을 20% 과소평가하는 경향이 있습니다." (계획 능력 향상)
4.  **Long-term Velocity (장기 속도)**
    - **기능:** 월간/연간 생산성 추이 그래프.
    - **데이터:** 총 업무 시간, Deep Work 비율의 장기적인 변화.
5.  **Data Export (블랙박스 추출)**
    - **기능:** 모든 플래너 데이터를 CSV/PDF로 추출하여 엑셀이나 노션에서 2차 가공 가능.

## 2. ⚙️ SETTINGS (설정) - "Cockpit Settings"

**목표:** 사용자가 자신의 환경을 완벽하게 통제할 수 있다는 느낌(Control)을 주어야 합니다.

### 👤 Identity (파일럿 설정)

- **Call Sign:** 사용자 이름 / 닉네임 설정.
- **Sync Status:** "Mothership Connected" (클라우드 동기화 상태 표시).

### 📐 Flight Systems (플래너 환경 설정)

- **Day Cycle Config (하루 주기 설정):**
  - _Grid Start/End Time:_ 현재 모달에서 하는 것뿐만 아니라, 기본 그리드 시작 시간을 04:00 AM으로 할지, 07:00 AM으로 할지 전역 설정.
- **Time Quantum (시간 단위):**
  - 기본 10분 외에 "5분(Hardcore)" 또는 "15분(Light)" 단위 선택 옵션.
- **Color Protocol (카테고리 색상 의미 설정):**
  - 파란색 = Work, 빨간색 = Critical 등 각 색상에 라벨을 붙여 통계의 정확도 향상.

### 🎨 Visuals (시각 효과)

- **Overlay Mode:** 드래그 시 보이는 '색칠'의 투명도 조절.
- **Theme:**
  - _Default (Musk White)_
  - _Dark Matter (OLED Black)_ - 밤 시간대 눈 보호.

### 🔔 Alerts (알림 및 경고)

- **Morning Briefing:** 매일 아침(예: 07:00) 플래닝 알림.
- **End of Day Review:** 하루 마감(회고) 알림.
- **Block Overrun:** 현재 시간이 설정된 블록을 초과했을 때 "지금 뭐하고 있습니까?" 같은 자극적인 푸시 알림.

### 💳 Access Level (구독 관리)

- **Upgrade to Pro:** Pro 기능 잠금 해제 버튼.
- **Guest vs Crew:** 로그인 관리.

## 3. 🖱️ Interaction & Logic Details (인터랙션 및 로직 상세)

마우스 클릭 및 터치 시 발생하는 상세 인터랙션 시나리오입니다.

### 📊 Stats Page Interactions

1.  **일일 완료율 (Daily Launch Readiness)**
    - **동작:** 원형 그래프 클릭.
    - **모달:** "일일 미션 체크리스트 (Daily Mission Checklist)"
    - **내용:**
      - 오늘 설정한 주요 과업(Main Thing) 리스트 표시.
      - 완료된 항목은 취소선 + 체크 아이콘 처리.
      - 미완료 항목은 일반 텍스트로 표시.
      - 하단에 "미완료 항목 내일로 미루기" 버튼 제공.

2.  **주간 모멘텀 (Weekly Momentum)**
    - **동작:** 막대 그래프 영역 클릭.
    - **모달:** "미션 로그 (Mission Log)"
    - **내용:**
      - 요일별 상세 몰입 시간 표시 (예: "월요일: 8.5시간 몰입").
      - 이번 주 총 생산 시간 대비 지난주 성능 비교 (예: "지난주 대비 15% 증가").

3.  **시간 할당 (Time Allocation)**
    - **동작:** 도넛 차트 영역 클릭.
    - **모달:** "자원 배분 현황 (Resource Distribution)"
    - **내용:**
      - 각 카테고리별 정확한 기록 시간(시간/분 단위) 리스트.
      - 전체 대비 차지하는 비중(%) 상세 표시.

4.  **뉴럴 링크 / Pro 기능 (Neural Link)**
    - **동작:** 잠겨있는 Pro 기능 카드 클릭.
    - **모달:** "뉴럴 링크 업그레이드 (Upgrade to Neural Link)"
    - **내용:**
      - 월 $5 구독 시 해제되는 고급 텔레메트리 기능 설명.
      - 정밀 히트맵 및 분석 도구 데모 이미지/설명.

### ⚙️ Settings Page Interactions

1.  **호출 부호 변경 (Identity)**
    - **동작:** 지휘관 이름("ELON") 클릭.
    - **반응:** 즉시 편집 가능한 텍스트 필드로 전환. 엔터 키 입력 시 저장.

2.  **하루 주기 설정 (Flight Systems)**
    - **동작:** "하루 주기 (Day Cycle)" 리스트 클릭.
    - **모달:** "하루 주기 설정"
    - **컨트롤:**
      - **기상 시간 (Grid Start):** 시작 시간 선택 (Picker 사용).
      - **취심 시간 (Grid End):** 종료 시간 선택 (Picker 사용).
      - _주의사항:_ "주기를 변경하면 플래너 그리드 뷰 전체가 이동합니다." 경고 문구 표시.

3.  **시간 단위 선택 (Time Quantum)**
    - **동작:** 시간 단위 리스트 클릭.
    - **선택 옵션:**
      - 5분 (하드코어)
      - 10분 (표준) - _기본값_
      - 15분 (라이트)
      - 30분 (휴가용)

4.  **색상 프로토콜 편집 (Color Protocol)**
    - **동작:** 리스트 아이템 클릭.
    - **모달:** "색상/카테고리 프로토콜"
    - **내용:**
      - 사용 중인 각 색상 팔레트와 라벨 편집 필드 나열.
      - 특정 색상을 어떤 업무(예: Blue -> 몰입 업무)로 부를지 직접 정의.

5.  **알림 설정 토글 (Alerts)**
    - **동작:** 스위치 클릭.
    - **반응:** 상태 즉시 반영 및 `localStorage` 저장. 토글 시 미세한 진동 피드백(지원 기기).

6.  **위험 구역 (Danger Zone)**
    - **데이터 내보내기:** 클릭 시 `mission_log.csv` 파일 생성 및 즉시 다운로드.
    - **데이터 삭제:** 클릭 시 빨간색 경고 모달 ("정말로 모든 데이터를 삭제하시겠습니까?"). 입력창에 "삭제"라고 타이핑해야만 삭제 버튼 활성화.
